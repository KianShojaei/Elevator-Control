A small **sample** of the dataset is provided with this repository as a compressed RAR file named `sample.rar` (attached to the release tagged `sample`). The archive contains ten randomly selected videos from the full collection together with their corresponding label files, so you can quickly inspect the data format and labels before downloading the entire set.


The complete dataset has been uploaded to Google Drive as **WholeDataSet.rar**. Please note that the full archive is **password-protected**; to receive the extraction password, email me at [kianbusiness18@gmail.com] and I will reply with the password and any additional usage notes. You can download the full dataset from the following [Google Drive link](https://drive.google.com/file/d/1DAB72Dr3ZO6bqvjpNMt2Z7SjYAfu-l5f/view?usp=sharing).
